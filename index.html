Para relaxar, um pouco de Markdown ...

Como eu estou usando o Hexo para desenvolver o meu Blog.

Este Blog que vos fala. O Markdown é uma ferramenta fundamental.

Por isso vou dedicar aqui algum tempo para aprender.

E, ao mesmo tempo compartilhar esse conhecimento com todos.

Criando um cabeçalho:

Cria-se um cabeçalho utilizando: O cactere #
Um caracter # para cada nível de cabeçalho
O níveis de cabeçalho vão de 1 a 6.
Ele(s) devem ser escritos a frente da linha que será formatada,
precedido de um espaço em branco.

Exemplos:

Cabeçalho de nível 1: # Texto

Este é um cabeçalho de nível 1

Cabeçalho de nível 2: ## Texto

Este é uma cabeçalho de nível 2

Cabeçalho de nível 3: ### Texto

Este é uma cabeçalho de nível

Cabeçalho de nível 4: #### Texto

Este é uma cabeçalho de nível 4

Cabeçalho de nível 5: ###### Texto

Este é uma cabeçalho de nível 5

Cabeçalho de nível : ####### Texto

Este é uma cabeçalho de nível 6

Vamos fazer uma série de posts para não ficar muito chato.

Felicidade a todos!
