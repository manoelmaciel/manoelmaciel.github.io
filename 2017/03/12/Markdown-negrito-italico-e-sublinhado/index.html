Markdown: itálico, negrito e enfatizado ...

Itálico *

Para marcar um texto como itálico devemos acrescentar um asterisco de cada lado do texto.

Exemplo: *texto*

Todo este texto está marcado como itálico.

Negrito **

Para marcar um texto como negrito basta acrescentar dois asterisco de cada lado do texto.

Exemplo: **texto**

Todo este texto está marcado como negrito.

Enfatizado __

Para marcar um texto como enfatizado é só envolvê-lo entre dois underlines.

Exemplo: __texto__

Todo este texto está marcado como enfatizado.

O resultado no código HTML tanto de negrito como de enfatizao e o texto envolvido pela tag strong.
